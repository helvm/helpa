## Another way to compilation

### Etlas

Compile and run with `etlas`:
```bash
etlas clean && etlas build && etlas test
etlas run helpa
```

### Gradle

Compile and run with `gradlew`:
```bash
./gradlew clean -PetaSendMetrics=true
./gradlew compileEta
./gradlew compileTestEta
./gradlew test
./gradlew run
./gradlew shadowJar
```
